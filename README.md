使用方式：（仅在linux环境下，其他环境如max or window需要自行修改部分config.sh)
	1	运行config.sh即可
	2	打开vim，输出BundleInstall命令。
		*	如果出现错误，请检查网络，对应的git仓库是否正确。
	3	创建session文件。指令:mks project.vim
		*	session命名可更改，但需要更改对应projectx.vim。
