# 项目定位：工具库。
## xls读取解析库。
### 当前进度。
	已可应用。通过几个简单的项目测试过。
### 已知缺陷。
	* 部分版本的xls会有小数点后无意义的零丢失。
	* 任何类型的数据都会转换成string保存。所以需要自行转换类型。
